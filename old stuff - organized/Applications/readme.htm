FlashForum 1.0 Readme

软件名称：FlashForum
  版本号：1.0
    作者：Huwell
编程语言: Flash,ASP
  数据库：Access
运行环境: Flash 4.0/5.0
软件大小：26KB
发行方式: 按GPL条款发布

这用Flash和ASP技术构建的BBS论坛.它不同于传统的基于Html的论坛.由于采用
了Flash技术,所以它是一款动态的Web论坛.

本软件是按照GPL条款发布的，目的只有一个： 采用开源的方式，使更多的人来
研究它的代码，从而让大家一起来维护和升级它：找出它的缺点,Bug；编写出更
优美的代码，算法；加入自己最炫的Flash效果等等。

一个人的力量是有限的，而大家的力量是无穷的。我希望在你的参与下，FlashForum会出
现：PHP版，Perl版，Flash 5.0版以及各种各样的效果版，音乐版等等。 最后，我真诚的
希望FlashForum能自由的生长。
